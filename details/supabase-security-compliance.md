# Supabase Security & Compliance

**Category:** Privacy  
**Tags:** privacy, GDPR

## Overview
- Central hub for information on how Supabase secures data and infrastructure.
- Focused on security posture, regulatory compliance, and privacy practices.
- Intended for security teams, compliance officers, auditors, and customers assessing Supabase’s risk profile.
