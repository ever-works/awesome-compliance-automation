# IcePanel Trust Center

## Overview
The IcePanel Trust Center provides information about IcePanel’s security, compliance, and privacy practices. It centralizes details on controls, certifications, and policies, and uses automated evidence and status sharing typical of modern compliance automation platforms.

- **Provider:** IcePanel
- **Category:** SOC 2 / Security & Compliance
- **Use Case:** Give customers and prospects visibility into IcePanel’s security posture and compliance status.

## Features
- **Centralized security & compliance hub** – Single location to review IcePanel’s security, privacy, and compliance information.
- **Compliance controls visibility** – Communicates the security and compliance controls IcePanel has in place.
- **Certifications & attestations overview** – Shares information on certifications (e.g., SOC 2) and other compliance frameworks supported by IcePanel.
- **Policies and documentation** – Provides access to formal security, privacy, and compliance policies.
- **Automated evidence sharing** – Leverages automation to share up-to-date evidence of controls, reducing manual back-and-forth.
- **Real-time / current status updates** – Uses automated status sharing so customers can see the current state of compliance and security posture.

## Related Product Context
IcePanel is a collaborative diagramming tool for software engineering and product teams, used to align on technical decisions. The Trust Center specifically addresses the security and compliance aspects of using IcePanel.
