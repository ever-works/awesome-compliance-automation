# Lumos Trust Center

**Website:** https://trust.lumos.com/

## Overview
Lumos Trust Center centralizes Lumos’ security and compliance posture, including documentation related to SOC 2 and privacy practices. It highlights how Lumos approaches security as an ongoing operational mindset and describes internal controls used to protect applications, permissions, and sensitive resources.

## Features

- **Centralized Security & Compliance Hub**  
  - Consolidates information on security controls, certifications, and policies.  
  - Includes documentation relevant to SOC 2 and related frameworks.

- **Security-as-a-Mindset Approach**  
  - Emphasizes security as an organizational mindset rather than a one-time checklist.  
  - Reflects both operational (process) and cultural (behavioral) security practices.

- **Least-Privilege Access Enforcement**  
  - Implements least-privilege access across critical resources, including credentials.  
  - Uses Lumos’ own platform internally to manage and enforce access policies.

- **Automated Access Requests**  
  - Uses Lumos access requests to control and document how users obtain access to applications and resources.

- **User Access Reviews (UARs)**  
  - Conducts Lumos user access reviews to periodically validate that granted permissions remain appropriate.

- **User Lifecycle Management**  
  - Manages access across the full user lifecycle (onboarding, role changes, offboarding) to maintain secure, up-to-date entitlements.

- **Shadow IT Detection**  
  - Detects unsanctioned or unmanaged SaaS applications in use within the organization.

- **Backchannel Access Detection**  
  - Identifies access to applications obtained outside of approved or sanctioned methods.

- **App Governance Focus**  
  - Oriented around governing SaaS apps and permissions at scale.  
  - Aims to support least-privilege and reduce unnecessary SaaS app spend through controlled access.

## Compliance & Standards

- **SOC 2 & Related Frameworks**  
  - Trust Center is positioned as a central place for SOC 2-related controls, documentation, and status.  
  - Supports privacy-focused practices aligned with modern security and compliance expectations.
